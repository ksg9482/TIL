### SQL 문법 순서
SELECT, FROM, WHERE, GROUP BY, HAVING, ORDER BY

### SQL 실행 우선 순위
FROM, ON, JOIN, WHERE, GROUP BY, HAVING, SELECT, DISTINCT, ORDER BY

### SQL 실행 순서
* 테이블 확인(FROM) 
* 조인 조건 확인(ON)
* 조건 맞게 테이블 병합(JOIN) 
* 데이터 추출 조건 확인(WHERE) 
* 특정 칼럼으로 데이터 그룹화(GROUP BY) 
* 그룹화 이후 테이터 추출 조건 확인(HAVING) 
* 데이터 추출(SELECT) 
* 중복 제거(DISTINCT) 
* 데이터 정렬(ORDER BY)
